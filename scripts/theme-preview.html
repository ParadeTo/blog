📝 使用方法

方式一：使用 npm script

# 默认主题（紫色）
npm run wechat source/_posts/your-article.md

# 指定主题
npm run wechat source/_posts/your-article.md -- --theme=blue
npm run wechat source/_posts/your-article.md -- --theme=orange

方式二：直接运行脚本

# 紫色主题（默认）
node scripts/md2wechat.js source/_posts/your-article.md

# 蓝色主题
node scripts/md2wechat.js source/_posts/your-article.md --theme=blue

# 橙色主题
node scripts/md2wechat.js source/_posts/your-article.md --theme=orange

转换后的使用流程

  1. 用浏览器打开生成的 HTML 文件
  2. 按 F12 打开开发者工具
  3. 在 Console 控制台输入：copyContent()
  4. 粘贴到微信公众号编辑器（Cmd+V / Ctrl+V）
