# 博客统计

扫描 `source/_posts/` 目录下的所有 `.md` 文件，输出以下统计信息：

1. **文章总数**
2. **总字数**（正文部分，不含 frontmatter）
3. **标签分布**：每个标签对应的文章数，按数量降序排列
4. **分类分布**：每个分类对应的文章数
5. **最近 5 篇文章**：标题 + 日期

输出格式为简洁的 Markdown 表格或列表，不需要额外的解释和建议。
